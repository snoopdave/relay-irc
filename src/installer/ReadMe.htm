Instructions for Running InstallAnywhere "Other Platforms" Installers

OS/2 Warp Instructions:

    Instructions

    • After downloading, type
        jre -nojit -cp relay08.zip install
    • Or you can type
         CLASSPATH=relay08.zip
        java -nojit install


    Notes

    • You must already have Java 1.1.6 or later installed. You can download Java from IBM's site
    • In a console window, change to the directory where you downloaded relay08.zip to before running the installer

All Other Platforms Instructions:

    Instructions (Unix or Unix-like operating systems)

    • After downloading, type
        jre -nojit -cp relay08.zip install
    • If that doesn't work, try
        jre -nojit -classpath [path to]classes.zip:relay08.zip install
    • If that doesn't work either, on sh-like shells, try
        cd [to directory where relay08.zip is located]
        CLASSPATH=[path to]classes.zip:relay08.zip
        export CLASSPATH
        java -nojit install
    • Or for csh-like shells, try
        cd [to directory where relay08.zip is located]
        setenv CLASSPATH [path to]classes.zip:relay08.zip
        java -nojit install


    Instructions (for other platforms)

    • You must already have Java 1.1 installed. You can download Java from Sun's site
    • In a console window, change to the directory where you downloaded relay08.zip to before running the installer
    • Your operating system may invoke Java in a different way. To start the installer, add relay08.zip to your CLASSPATH, then start the main class of the installer named install
